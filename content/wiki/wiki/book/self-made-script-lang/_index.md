## 2.术语

| 英文 | 中文       | 解释                    |
| ---- | ---------- | ----------------------- |
| id   | 标识符     | 指 identifier           |
| str  | 字符串     | 指 string               |
| pat  | 模式       | 指 pattern              |
| num  | 数字       | 指 int 类型的 number    |
| que  | 队列       | 指 queue                |
| idx  | 索引下标   | 指 index                |
| lex  | 词法分析器 | 指 lexer                |
| ast  | 抽象语法树 | 指 abstract syntax tree |
| expr | 表达式     | 指 express              |
