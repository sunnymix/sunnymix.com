MacPorts 原名 DarwinPorts，是 macOS 和 Darwin 操作系统的软件包管理系统，用于简化软件的安装和管理。

## 官方资料

- [MacPorts Guide](https://guide.macports.org/)
- [MacPorts Ports](https://ports.macports.org/)

## 常用命令

### search 搜索

```
$ port search <package-name>
```

### install 安装

```
$ sudo port install <package-name>
```
