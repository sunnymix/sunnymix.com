## 安装配置（macOS）

- 官网下载安装包：https://julialang.org/downloads/
- 默认启动方式：mac / 应用程序 / Julia-1.7.app
- 配置终端命令：alias julia="exec /Applications/Julia-1.7.app/Contents/Resources/julia/bin/julia"
- 第一个 Julia 程序：

```
julia> "Hello," * "World"
"Hello,World"
```
